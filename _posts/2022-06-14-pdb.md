How to test our program without print? And what can we do to select the exact problem from our program?

```ruby
import pdb; pdb.set_trace() # set the stop point
```
if you see the following content in command line. Then we reach the stop point
```ruby
(Pdb)
```
And we have several useful command here in `(Pdb)` terminal:
```ruby
l               # 查看上下 11 行代码
ll              # 查看整个文件的代码
s               # 执行下一行，进入function
n               # 执行下一行，不进入function
r               # 执行下一行，如果在function中，执行到return line
c               # 执行到下一除断点
unt lineno      # 执行到指定行
j lineno        # 跳到后面的行，被跳过的代码不执行
a               # 查看parameters和参数的值
q               # 退出pdb
```
Of course, there are several methods of using non-invaisve method to plug `pdb` inside. It can be refered to the pdb Documentation [pdb-Documentation].

[pdb-Documentation]: https://docs.python.org/3/library/pdb.html
